About Chat Server/Chess

Chat Room Functionality

This chat server is a real-time communication platform that enables users to join different chat rooms and exchange messages instantly. Created as a collaborative effort, it demonstrates the capabilities of modern web technologies.

Features:

  • Multiple chat rooms
  • Real-time messaging
  • User anonymity and privacy
  • Simple and intuitive UI

Chess Functionality

This chess platform provides an interactive and engaging gaming experience, allowing users to play against each other or against an AI. Utilizing WebSocket technology, the platform ensures real-time updates and responsive gameplay.

Interactive Chessboard:

  • Click-to-Move Functionality: Players can select a chess piece by clicking on it. Once selected, the system highlights all possible moves based on chess rules, aiding strategic planning.
  • Highlighting Possible Moves: When a piece is selected, the chessboard visually indicates all legal moves for that piece.
  • Enforces Chess Rules

Game Features:

  • Player vs. Player
  • Real-time game updates through WebSocket communication

This chess game was developed to demonstrate advanced web technologies and real-time data handling
