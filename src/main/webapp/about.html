About Chat Server

This chat server is a real-time communication platform that enables users to join different chat rooms and exchange messages instantly. Created as a collaborative effort, it demonstrates the capabilities of modern web technologies.

Features:

  • Multiple chat rooms
  • Real-time messaging
  • User anonymity and privacy
  • Simple and intuitive UI

Creators:

  • Smit Kalathia
  • Bav Sandhu
  • Bohdan Synytskyi
  • Nimreet Gill

This project was created for the course Software Systems Dev. & Integ. - CSCI-2020U at Ontario tech University. It provides an understanding of web sockets, server-client communication, and user interface design.
